# 使用11ty写博客

## Install 
```bash
yarn
# or
pnpm i
```

## Build
```bash
yarn build
# or
pnpm build
# or
eleventy --output=docs
```

## Run
```bash
yarn serve
# or
pnpm serve
# or
eleventy --serve  --output=docs
```


Then access the site with the URL [http://localhost:8080/](http://localhost:8080/) (your port may vary).
